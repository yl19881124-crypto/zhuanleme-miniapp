# 赚了么小程序 V2 总方案（单文件）

> 只看这个文件就够了。

## 1. 本期目标（4 周）
- 统一 Stitch UI 风格
- 跑通主流程：首页 -> 记账 -> 结算
- 保证数据口径一致（收入/支出/净收入/时长）

## 2. 范围边界
### 本期做
- 首页：今日已赚、净收入、状态切换、快速记账
- 记录：消费列表与基础编辑
- 结算：今日战报和时长结构
- 设置：时薪与时间配置、隐私配置

### 本期不做
- 海报生成（后续 image2 独立立项）
- 社交排行、复杂报表

## 3. 里程碑
- 第1周：方案冻结 + 组件规范
- 第2周：首页/记账链路
- 第3周：记录/结算
- 第4周：设置收口 + 灰度修复

## 4. 验收标准
- 30 秒内完成：首页 -> 记账 -> 结算
- 视觉规范一致，不出现临时样式
- 设置修改后数据立即生效

## 5. 执行原则
- 先方案后开发
- 每周只推进 1~2 个核心目标
- 每阶段有明确“做什么/不做什么/怎么验收”
